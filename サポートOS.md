### サポート OS

| OS            | runs-on キーに指定できるバージョン              |
| ------------- | ----------------------------------------------- |
| Linux(Ubuntu) | ubuntu-latest・ubuntu-22.04・ubuntu-20.04 など  |
| Windows       | windows-latest・windows-2022・windows-2019 など |
| macOS         | macos-latest・macos-14・macos-13 など           |
